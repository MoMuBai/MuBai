## First
#### Android屏幕适配：
使得某一元素在Android不同尺寸、不同分辨率的手机上具有相同的显示效果

##### 为什么需要屏幕适配：
* Android系统碎片化：各种系统，小米、华为...（自己定制化了很多）  
* 屏幕尺寸碎片化：5.0、5.5、6.0...  
* 屏幕分辨率碎片化:230x480、720x1080、1920x1080...
  
##### 具体方法：  
* 布局适配（RelativeLayout、LinearLayout的选用）  
* 根据屏幕的配置来加载相应的UI布局  
* dp替换px  
* 百分比布局
